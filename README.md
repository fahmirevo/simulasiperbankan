# simulasiperbankan
Simulasi perbankan menggunakan java

Tools :

Java FX
RMI

How to use :

compile all

    javac *.java

then start the registry

Windows

    start rmiregistry

Linux

    rmiregistry

then put this command
each line on different terminal

    java ServerApplication
    java Teller <number of teller>
    java ATMApplication
    java TellerApplication
